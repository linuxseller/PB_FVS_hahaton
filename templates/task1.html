Задание 1.

На работу имеется N кандидатов, но осталось только vacancies свободных мест. Необходимо выбрать именно столько лучших кандидатов, при равных баллах отдавая предпочтение кандидату с меньшим ID.

По GET запросу, сервер выдает JSON в формате:

{
    "vacancies": <int>,
    "applicants": [
          {
            "name": <str>,
            "uuid":<str>,
            "git":<bool>,
            "project_experience":<bool>,
            "free_time":<int>
          },
          ...
      ]
}

Баллы кандидатам распределяются по следующим критериям наличие Git <=> 10 points опыт в проектах <=> 20 points сколько часов в неделю готов работать X <=> | X < 20 = 0 | 20 <= X < 40 = 10 points | x >= 40 = 20 points

Для проверки необходимо предоставить JSON в формате:

{
    "best": [
        {
            "name": <str>,
            "uuid": <str>,
            "score": <int>
        }
        ...
    ]
}
