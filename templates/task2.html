Задание 2.

У менеджеров на работе очень много задач, и от них поступило ТЗ разработать автоматический ежедневник, который отбирает из списка задач такие, что бы успеть выполнить их максимальное количество за рабочий день. Для теста попробуйте распределить сначала задачи разработчика в течение его рабочего дня.

По GET запросу, сервер выдает JSON в формате:

WARNING: priority temporary is not considered and removed from actual API!
{
    "tasks": [
          {
            "title": <str>,
            "id": <int>,
            "duration": <int>,
            "deadline": <str: f"hh:mm">,
            "priority": <int>
          },
          ...
    ]
}
priority = range[1,20]
priority 20 => very important
priority 1 => not important

Считайте начало рабочего дня в 9:00, и конец в 19:00 без обеденного перерыва.

Для проверки необходимо предоставить JSON в формате:


{
    "schedule": [<int>, ...]
}
