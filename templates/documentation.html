REST API сервера

Получение ТЗ

GET /hahaton/easy/task/text/<taskid>

Рекомендуется открывать в браузере.

Получение данных для выполнения ТЗ

GET /hahaton/easy/task/<taskid>

Отправка решения для валидации

POST /hahaton/easy/task/<taskid>

| название поля | тип поля |                         описание                          |
|---------------|----------|-----------------------------------------------------------|
|   solution    |   json   | Поле содержит JSON формата соответствующего требованию ТЗ |
|     team      |  string  | Поле содержащее название команды, не рекомендуется менять |

Примеры создания запросов к серверу с использованием python

Для получения информации (GET requests)

import requests as rq

url = ...
r = rq.get(url)
print(r.text) # .text -- поле хранящее в себе текст ответа сервера

Для отправки данных (POST requests)

import requests as rq

url = ...
payload = {'team': 'FVS_No1', 'solution': '{"answer": 10}'}
r = rq.post(url, data=payload)
print(r.text)
